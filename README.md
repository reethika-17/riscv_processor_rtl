# RISC-V Processor RTL

## Overview

This repository contains the RTL implementation and verification of key modules of a single-cycle RISC-V processor.

The project focuses on instruction decoding, memory access, branching, jumping, register file operations, and firmware execution using Verilog HDL.

---

## Features

- Instruction Fetch Unit (IFU)
- Register File
- Load Unit
- Store Unit
- Branch Unit
- Jump Unit
- Memory Interface
- Immediate Decoder
- Instruction Decoder
- ALU Core
- Instruction/Data Arbiter

---

## Verification

The RTL modules were verified using:

- Cocotb
- Python Testbenches
- Icarus Verilog

Firmware examples were also executed for functional validation.

---

rtl/
